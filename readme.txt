Quick Start Guide
-----------------
1. run lumped_model_inputs.m in matlab
2. select desired outputs (click "Basic parameters" to select Pressure LA, LV, SA, Volume LA, LV and Velocity MV, AV)
3. select run and graphs should appear with the desired outputs
4. click "close all" to close all graphs
5. click "save all" to save all graphs including input parameter (GUI panel)
